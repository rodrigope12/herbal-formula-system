# Upwork Project Submission: Herbal Formula System

## Project Title
**Clinical-Grade Herbal Formulation Engine**

## Catchy Tagline
**"Pharmacological Precision. Aesthetics of Apple."**

## Category
**Development & IT** > **AI & Machine Learning** (or **Web Application Development**)

## Project Description
A high-performance algorithmic platform developed to automate personalized botanical prescriptions based on clinical safety data. This system transforms traditional herbal wisdom into a precise, medical-grade formulation tool.

**Technical Pillars:**
- **Clinical Safety Guardrails:** Dynamic enforcement of contraindications (Pregnancy, Allergies, Polypharmacy).
- **Synergy Optimization Algorithm:** Scoring engine that maximizes blend efficacy through positive interaction weighting.
- **Precision Dosing:** Percentage-based dosage calculation with milligram accuracy for professional compounding.
- **Premium UX/UI:** Distraction-free clinical interface built with Streamlit and bespoke CSS architecture.

**Tech Stack:**
- **Core:** Python 3.9+, Pandas
- **Frontend:** Streamlit (High-End CSS Styling)
- **Data:** JSON-driven Clinical Knowledge Base
